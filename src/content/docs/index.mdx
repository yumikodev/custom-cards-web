## Sobre nosotros

Este proyecto fue creado por [**Yasu Yumiko**](https://yumikodev.vercel.app) (Edwin J.) el 9 Octubre 2022 con la finalidad de poder crear imágenes personalizadas para usarse en bots de Discord. Un sueño bastante genial para él, pode empaquetar su código y que otros puedan usarlo con la mayor libertad posible como si lo estuvieran haciendo desde cero, pero quitando la complejidad que realmente hay por debajo.

Aunque todo pudo mantenerse en un simple NPM, al explorar otros entornos de JavaScript (Bun, Deno, etc) notó la incompatibilidad con las dependencias del proyecto y que incluso estas podían darse en sistemas fuera de Linux. Su solución fué mover toda esta complejidad a una API y cambiar la generación de las tarjetas en solicitudes `POST` a un endpoint.

Todo funcionaba bien al inicio, pero lo que complicó las cosas fue su carencia económica para pagar un buen hosting 24/7 y su falta de conocimientos para mantener una buena API. (Tenía 14 años en ese entonces y aún con 16, a la fecha que escribo esto, me falta dinero 🥲)

Después de casi 3 años (14 Enero 2025), se lanzó oficialmente la versión 2 de la API que trajo muchas novedades estupendas y un plan para poder hacer mantenible este proyecto.
